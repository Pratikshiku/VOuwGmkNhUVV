### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、教师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 计算机专业相关的软件
5. 课程
6. 音乐
7. 个人中心
8. 计算机专业相关的软件收藏
9. 课程收藏


**教师功能：**
1. 登录
2. 修改密码
3. 个人中心
4. 计算机专业相关的软件管理、计算机专业相关的软件留言管
5. 课程管理、课程留言管理
6. 宿舍管理、宿舍人员管理、宿舍归寝管理、宿舍报修管理
7. 音乐管理
8. 成绩管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 基础数据管理（班级管理、备忘录类型管理、记事本类型管理、软件类型管理、课程类型管理、音乐类型管理）
4. 记事本管理
5. 备忘录管理
6. 计算机专业相关的软件管理、计算机专业相关的软件留言管理、计算机专业相关的软件收藏管
7. 课程管理、课程留言管理、课程收藏管理
8. 老师管理
9. 宿舍管理、宿舍人员管理、宿舍归寝管理、宿舍报修管理
10. 音乐管理、音乐留言管理、音乐收藏管理
11. 用户管理
12. 成绩管理
13. 轮播图信息
